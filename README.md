# native-mailer
Little Email client demo app for OSX

## How to run it?
Install `react-native-macos`

```
npm install -g react-native-macos-cli
```

Install node packages:

```
npm install
```

Open the Xcode's project:

```
cd react-native-mailer
open macos/Mailer.xcodeproj
```
